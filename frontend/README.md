# JoByte Frontend

React + TypeScript + Vite frontend for JoByte.

## Scripts
- `npm run dev` - start local dev server
- `npm run build` - type-check and production build
- `npm run lint` - run ESLint checks
- `npm run preview` - preview production build locally

## App Areas
- Landing page
- Auth and role selection page
- Candidate dashboard
- Employer dashboard
- Job listing and unlock modal flow

## Structure
- `src/components/` reusable UI primitives
- `src/pages/` route-level page components
- `src/constants/` route and configuration constants
- `src/data/` frontend mock data
- `src/types/` shared frontend types

## Environment
Copy `.env.example` to `.env` and set local values.
