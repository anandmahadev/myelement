# Security Policy

## Supported Versions
JoByte is currently pre-release. Security fixes are applied on `main`.

## Reporting a Vulnerability
1. Do not open a public issue.
2. Report privately with reproduction steps and impact.
3. Include affected route(s), payloads, and screenshots if possible.

## Secret Management
- Never commit real API keys or tokens.
- Use environment variables and provide `.env.example` templates.
- Rotate exposed keys immediately.
