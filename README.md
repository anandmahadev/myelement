# JoByte

JoByte is a hiring platform prototype that connects applicants and employers with a structured, assessment-first workflow.

## Repository Structure
- `frontend/` - React + TypeScript + Vite client app
- `backend/` - backend service workspace
- `API_SPEC.md` - source-of-truth contract for service interfaces

## Current Scope
- Marketing and entry experience (landing + auth)
- Candidate dashboard shell
- Employer dashboard shell
- Job discovery page with filtering UI

## Getting Started
1. Go to `frontend/`
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

## Contract-First Rule
Before changing API payloads or routes, update `API_SPEC.md` first.
